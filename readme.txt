Archive content and installation.
---------------------------------
1. "res_mods" folder- basic config. Required to install (copy the folder to ..\World_of_Tanks\).
2. "fonts" folder- fonts used in the config. Required to install (install fonts from the folder to c:\Windows\Fonts\).
3. "_colorBlind" folder- enabling color blind mode. Install if necessary after the basic config installation (copy the folder content to ..\World_of_Tanks\). The color blind mode touchs on the battle part completely and the hangar part regarding statistics colors only.
4. "_sixthSense" folder- "sixth sense" skill lamps: v.1 "flare", v.2 "danger sign" (general or for color blind). Install if necessary after the basic config installation (copy the chosen version folder content to ..\World_of_Tanks\). The default state in the basic config is v.1 "flare", so you don't need to install this one if this is your choise.

  Note: tank icons with vehicle names are required for config.

Support & updates: http://www.koreanrandom.com/forum/topic/2172-


Содержимое архива и установка.
------------------------------
1. Папка "res_mods"- оснофной корфиг. Обязательно к установке (папку скопировать в ..\World_of_Tanks\).
2. Папка "fonts"- шрифты используемые в конфиге. Обязательно к установке (установить шрифты из папки в c:\Windows\Fonts\).
3. Папка "_colorBlind"- включение режима цветовой слепоты. Устанавливается по необходимости поверх основного конфига (содержимое папки скопировать в ..\World_of_Tanks\). Режим цветовой слепоты затрагивает боевую часть полностью, а ангарную только касаемо цветов статистики.
4. Папка "_sixthSense"- лампочки умения "6 чувство": v.1- "блик", v.2- "знак опасности" (обычная или для цветовой слепоты). Устанавливается по необходимости поверх основного конфига (содержимое папки с выбранной версией скопировать в ..\World_of_Tanks\). Состояние по умолчанию в основном конфиге- v.1 "блик", если это Ваш быбор, то он не требует отдельной установки.

  Важно: для конфига обязательны иконки танков с названиями техники.

Поддержка и обновления: http://www.koreanrandom.com/forum/topic/2172-
